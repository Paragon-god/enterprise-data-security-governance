## LinkedIn Project Title

Enterprise Data Security Governance and Technical Controls Implementation

## Description

Designed and implemented a security governed enterprise analytics prototype for Northstar Retail Group, a simulated retail organisation.

The project connected Spark based data processing, Parquet storage, R modelling, Shiny dashboards, MongoDB analytical serving, and a basic query application with a complete GRC layer.

Produced a data inventory, risk register, treatment plan, 20 control catalogue, access matrix, RACI, retention schedule, privacy assessment, model risk record, evidence register, and management recommendations.

Validated data integrity controls, anomaly analysis, secret separation, TLS protected database access, model performance disclosure, human oversight, and sanitised public evidence.

The prototype is suitable for portfolio demonstration. Production approval remains withheld pending enforced access roles, restricted networks, authenticated applications, central monitoring, tested recovery, approved retention, vendor review, incident exercises, and independent model validation.

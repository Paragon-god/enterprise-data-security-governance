# Dataset Notice

The source dataset is deliberately excluded from this repository.

Use only a lawfully obtained dataset that you are authorised to process.

The original prototype used a retail transaction dataset with 9,994 records and a simulated website event dataset with 100 records.

Do not commit:

* customer data;
* behavioural identifiers;
* live exports;
* restricted internal datasets; or
* data that you do not have permission to publish.
